# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in CPS, please report it through
the ONAP vulnerability management process:

https://lf-onap.atlassian.net/wiki/spaces/DW/pages/16351513/ONAP+Vulnerability+Management

Do NOT open a public Gerrit change or mailing list post for security issues.
